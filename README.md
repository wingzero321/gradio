# 股票预测应用

基于LSTM的股票价格预测系统，可以：
- 预测单只股票未来3天走势
- 批量预测多只股票
- 显示历史数据对比
- 计算预期涨跌幅

## 使用方法
1. 选择股票代码
2. 点击预测按钮
3. 查看预测结果和历史数据对比

## 功能特点
- 支持单股票预测和批量预测
- 显示7天历史数据
- 预测未来3天走势
- 计算预期涨跌幅
- 数据库存储预测结果
